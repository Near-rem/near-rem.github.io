# 🛠️ Near Rem - Arsenal | Dashboard OSINT & Web Sec

![Status](https://img.shields.io/badge/Status-Development-red?style=for-the-badge)
![Role](https://img.shields.io/badge/Role-Pentester%20%2F%20Full%20Stack-blue?style=for-the-badge)

O **Near Rem - Arsenal** é um dashboard centralizado de ferramentas para Inteligência de Fontes Abertas (OSINT), análise de redes e auditoria de segurança web. Este projeto foi desenvolvido para otimizar o fluxo de coleta de dados e perfilamento de alvos em ambientes controlados.

> **AVISO:** Este projeto tem fins puramente educacionais e de pesquisa. O uso indevido dessas ferramentas para atividades ilícitas é de total responsabilidade do usuário.

---

## 🚀 Funcionalidades Principais

* **Network Intelligence:** Rastreamento de IP, consulta de provedores (ISP) e geolocalização em tempo real.
* **Hardware Profiling:** Extração de metadados de hardware (GPU, CPU Threads, RAM, Bateria).
* **OSINT Arsenal:** Atalhos rápidos para bases de dados judiciais, vazamentos (Leaks) e Darknet.
* **Social Engineering Layer:** Interface personalizada para interação e coleta passiva de dados de acesso.

---

## 🛠️ Tecnologias Utilizadas

* **Frontend:** HTML5, CSS3 (Custom Cyberpunk/Terminal Theme), JavaScript (Vanilla).
* **APIs de Terceiros:** IPAPI, IPWhoIs, DNSDumpster.
* **Hardware Extraction:** WebGL API, Battery Status API.

---

## 📁 Estrutura do Projeto

* `index.html`: Terminal de comando central.
* `arsenal.html`: O hub de ferramentas web e manuais.
* `extrair.html`: Dashboard de extração de dados do visitante.
* `sobre.html`: Perfil técnico do desenvolvedor.
* `contatos.html`: Links de conexão segura.
